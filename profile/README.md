# CNU 똑똑이: 키워드 기반 게시글 태그 분류 및 알림 서비스


<p>학내 게시판의 글을 키워드를 기반으로 분류하여 태그별로 게시글을 보여주고, 추가적으로 알림 서비스를 제공하는 게시판 앱이다.</p>
<p>교내 공공데이터 'CMS 홈페이지 게시판 등록 데이터'를 활용하였다.</p>
